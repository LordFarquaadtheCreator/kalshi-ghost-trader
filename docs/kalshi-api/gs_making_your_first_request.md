# Making Your First Request

> Start trading with Kalshi API in under 5 minutes

To make your request we recommend testing a public endpoint like [GetMarkets](api-reference/market/get-markets). As you explore our other endpoints, you'll notice some endpoints return an authentication_error. If you want to experiment with these endpoints, you will need to get [API Keys](gs_api_keys.md). You may also want to sign up for a demo account to test without real funds.

The following resources might help you on your journey to exploring Kalshi's markets:

- [**Quick Start: Market Data**](gs_quick_start_market_data.md)
- [**Quick Start: Authenticated Requests**](gs_quick_start_authenticated_requests.md)
- [**Discord**](https://discord.gg/kalshi) and check out #dev and #support
